# Car Detailing Academy

This repository contains a static GitHub Pages business website for Car Detailing Academy.

Car Detailing Academy sells digital downloadable car detailing guides, PDF manuals, checklists, worksheets, and educational resources. The website is intentionally simple, public, and suitable for business verification use.

## Files

- `index.html` contains the page content, product descriptions, delivery terms, refund policy, terms of use, privacy policy, and contact details.
- `styles.css` controls the responsive layout and visual design.
- `.gitignore` excludes common local operating system and editor files.

## Editing Products and Contact Details

To update product names, prices, descriptions, or delivery wording, edit the product cards in `index.html`.

To update the contact email, replace `8yymxf6vy4@privaterelay.appleid.com` wherever it appears in `index.html`.

To update design colours, spacing, or layout, edit `styles.css`.

## Enable GitHub Pages

1. Open the repository on GitHub.
2. Go to **Settings**.
3. Open **Pages**.
4. Set **Source** to **Deploy from branch**.
5. Set **Branch** to **main**.
6. Set **Folder** to **/root**.
7. Click **Save**.

The public GitHub Pages URL should be:

```text
https://thsk79vk85-hash.github.io/car-detailing-academy/
```

Once GitHub Pages is live, this URL can be submitted to Stripe as the business website for Car Detailing Academy.
